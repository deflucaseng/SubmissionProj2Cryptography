To run this project, run the following commands:

> pip install flask
> pip install joblib
> pip install pandas
> pip install sklearn

After you have installed these packages, run this command:

> python3 app.py

Once the server is up, you can access the web application on http://127.0.0.1:8000

NOTE: Make sure that no other application/service is running on port 8000.
If you have any other essential service running on port 8000, change the port in the last line of app.py file accordingly.
